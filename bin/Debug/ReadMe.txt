VanishClip은 단일 클립보드는 여러 데이터를 복사할 수 있게 하는 멀티 클립보드 프로그램 입니다.

****사용법****

컴퓨터를 Vanishclip.exe 파일을 시작프로그램 폴더에 넣어 주세요
프로그램이 실행되면 복사가 이루어질때마다 화면 우측 상단에 복사된 데이터가 표시됩니다.
복사된 데이터는 총 10개까지만 리스트화되어지고 추가적인 데이터는 첫번째가 없어지며 밀려납니다.
총 10개의 데이터는 Ctrl + 1 키부터 Ctrl + 0 키까지 순차적으로 단축키가 정해져 있습니다.
원하는 곳에 붙여넣기를 할때 해당 리스트의 인덱스를 가지고 붙여넣기 하면 됩니다.
또한 Ctrl + ` 키를 누르게 되면 우측상단에 리스트가 없어지지 않고 계속 보여지게 됩니다.
이 상태로 원하는곳에 키보드커서를 둔후 리스트의 데이터를 클릭하게 되면 해당 리스트의 데이터가 붙여넣기 됩니다.
또한 오른쪽 마우스를 클릭할 경우 해당 데이터가 삭제됩니다.
옵션창은 보자마자 아!! 하실겁니다.
